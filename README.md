**InVidious Project** is a Unity based multiplayer horror game that demonstrates a modular and scalable architecture using Netcode for GameObjects, Relay, and Lobby systems.

The project focuses on:

* Client-server synchronization
* NetworkVariable and RPC-based state management
* Role assignment and authority handling
* Modular gameplay systems (movement, inventory, items, ritual mechanics)

Key Features:

Asymmetric multiplayer (Exorcist vs Ghost)
Role-based abilities and gameplay mechanics
Networked player movement, animation, and interaction
Inventory and item-based ritual system
Modular and scalable architecture using Unity Netcode

Built as both a learning project and a foundation for advanced multiplayer systems.
